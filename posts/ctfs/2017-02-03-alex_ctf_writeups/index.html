CTF

Scripting

SC1: Math bot

This one was fairly easy. I started with trying and finding out, which input leads to which output.

When I gave the correct answer to a calculation, I immediately received the second question. After a while I started to automate my manual effort and writing the script ‘alex_ctf_scripting_solver.py’ using pwntools. As I didn’t know, how many questions I will face, I tested with 10 questions increasing to 20 afterwards. When I increased the number of questions to 1000, I finally got to the last question at 250. Now I just needed to adjust my script to print out the rest of the stream. Et voila! 

from pwn import *

r = remote('195.154.53.62',7331)

for i in range(1,251):
    r.recvuntil('Question  %d :\n' % i)
    calc = r.recvuntil('=', drop=True)
    print '%d %s' % (i , str(eval(calc)))
    r.send(str(eval(calc)) + '\n')

print r.recvall()
Note: Never use eval in productive coding !!!
